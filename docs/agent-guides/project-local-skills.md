# Project-Local Skills

Project-local Codex skills live in `.codex/skills/`.

Use them for repeatable development workflows:

- `sql-validator`: SQL safety and validator changes
- `bi-metric`: governed metric definitions and metric SQL behavior
- `workflow-node`: LangGraph-style node and workflow wiring
- `prompt-engineering`: runtime prompt edits
- `metadata-retrieval`: schema and semantic metadata flow
- `evaluation`: tests, regression cases, and evaluation questions
- `database-setup`: DDL, views, indexes, data loading, and read-only DB setup

These skills are for coding agents working on the repository. They are not runtime BI copilot prompts.
